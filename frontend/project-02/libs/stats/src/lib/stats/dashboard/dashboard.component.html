Basic Bar Chart Bar charts are used to compare the values of different categories. The length of the bar represents the value of each category.

The config of the bar chart is similar to the line chart. We just need to change the type in series to bar. In this way, we can get a simple bar chart.

Basic Scatter Chart Scatter charts are used to show the relationship between two variables. Each point represents a pair of values.

The config of the scatter chart is similar to the line chart and bar chart. We just need to change the type in series to scatter. In this way, we can get a simple scatter chart.
