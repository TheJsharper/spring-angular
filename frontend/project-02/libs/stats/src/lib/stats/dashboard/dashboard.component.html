Basic Bar Chart Bar charts are used to compare the values of different categories. The length of the bar represents the value of each category.

The config of the bar chart is similar to the line chart. We just need to change the type in series to bar. In this way, we can get a simple bar chart.

Basic Scatter Chart Scatter charts are used to show the relationship between two variables. Each point represents a pair of values.

The config of the scatter chart is similar to the line chart and bar chart. We just need to change the type in series to scatter. In this way, we can get a simple scatter chart.

Basic Candlestick Chart Candlestick charts are used to show the price movement of a financial instrument over time. Each candlestick represents the open, high, low, and close prices for a specific period.

The config of the candlestick chart is not completely the same as the line chart and bar chart. There is no need to configure the axis, but we need to set the min and max values in series to define the range of the candlestick chart.

Basic Radar Chart Radar charts are used to show the performance of multiple variables in a single chart. Each axis represents a variable, and the distance from the center represents the value of each variable.

The config of the radar chart is not completely the same as the line chart and bar chart. There is no need to configure the axis, but we need to set the shape in series to radar to define the type of the chart.

Basic Boxplot Chart Boxplot charts are used to show the distribution of a dataset. The box represents the interquartile range, the line inside the box represents the median, and the whiskers represent the minimum and maximum values.

The config of the boxplot chart is not completely the same as the line chart and bar chart. There is no need to configure the axis, but we need to set the shape in series to boxplot to define the type of the chart.

Basic Heatmap Chart Heatmap charts are used to show the relationship between two variables in a matrix format. The color of each cell represents the value of the relationship between the two variables.

The config of the heatmap chart is not completely the same as the line chart and bar chart. There is no need to configure the axis, but we need to set the shape in series to heatmap to define the type of the chart.
